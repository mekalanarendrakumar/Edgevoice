# EdgeVoice: Real-Time MFCC Accelerator UI

This project provides a real-time web interface for the EdgeVoice MFCC hardware accelerator, featuring:
- Real-time waveform and MFCC heatmap visualization
- Modern maroon/purple color scheme with lighting effects
- File upload and statistics display

## Tech Stack
- React
- Tailwind CSS
- Plotly.js

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Project Structure
- `src/` - React components and styles
- `public/` - Static assets
